# Gradient Descent Animation in Machine Learning

## 📌 Project Overview

This project demonstrates the working of **Gradient Descent**, one of the most important optimization algorithms in Machine Learning.

Using a simple linear regression problem generated with Scikit-Learn's `make_regression`, the notebook visualizes how the **slope (m)** and **intercept (b)** are updated during each iteration. The animation helps build an intuitive understanding of how Gradient Descent gradually minimizes the cost function and finds the best-fit regression line.

---

## 🎯 Objectives

* Understand the intuition behind Gradient Descent
* Learn how model parameters are updated iteratively
* Visualize the optimization process

* Understand cost function minimization

* Compute gradients for m and b

## 🔍 Key Observations

* Gradient Descent gradually reduces prediction error.

* Proper parameter updates lead to the best-fit regression line.
